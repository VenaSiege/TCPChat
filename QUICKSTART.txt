TCP 聊天软件 - 快速开始指南
========================================

【第一步】检查 Python 环境
----------------------------------------
- 确保已安装 Python 3.6 或更高版本
- 打开命令行，输入 python --version 查看版本
- 如未安装，请从 https://www.python.org 下载安装

【第二步】启动服务器
----------------------------------------
Windows 用户（任选一种方式）：

方式 1 - 批处理脚本：
  1. 双击 run_server.bat 文件
  2. 服务器启动，等待客户端连接

方式 2 - PowerShell 脚本：
  1. 右键点击 run_server.ps1
  2. 选择"使用 PowerShell 运行"
  3. 服务器启动

方式 3 - 命令行：
  1. 打开命令行/PowerShell
  2. 进入项目目录
  3. 输入 python server.py
  4. 服务器启动

【第三步】启动客户端
----------------------------------------
打开另一个/多个命令行窗口，重复如下步骤：

方式 1 - 批处理脚本：
  1. 双击 run_client.bat 文件
  2. 输入服务器地址（按 Enter 使用默认 localhost）
  3. 输入服务器端口（按 Enter 使用默认 5000）
  4. 输入用户名
  5. 开始聊天

方式 2 - 命令行：
  1. 打开命令行/PowerShell
  2. 进入项目目录
  3. 输入 python client.py
  4. 按照提示操作

【第四步】开始聊天
----------------------------------------
- 在客户端输入消息，按 Enter 发送
- 消息会自动发送给所有在线用户
- 输入 exit 退出客户端

【常见问题】
----------------------------------------

Q1: 无法连接到服务器？
A: 确保服务器已启动，且输入的地址和端口正确

Q2: 中文显示乱码？
A: 在 Windows 命令行中运行 chcp 65001

Q3: 端口已被占用？
A: 修改 server.py 中的端口号，或关闭已运行的服务器

Q4: 需要跨网络通信？
A:
  - 服务器客户端在同一局域网：输入服务器的局域网 IP
  - 服务器在公网：输入服务器的公网 IP 并配置防火墙

【网络配置示例】
----------------------------------------

本地测试（同一台机器）：
  服务器：localhost 或 127.0.0.1，端口 5000
  客户端：localhost 或 127.0.0.1，端口 5000

局域网通信（不同机器）：
  1. 在服务器机器上运行 server.py
  2. 查看服务器的局域网 IP（如 192.168.1.100）
  3. 在客户端机器输入服务器的局域网 IP
  4. 端口保持 5000
