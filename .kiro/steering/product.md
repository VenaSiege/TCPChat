# Product Overview

A simple TCP-based chat application that enables real-time multi-client communication. The system consists of a server that handles multiple concurrent connections and broadcasts messages to all connected clients.

## Core Features

- Multi-client support with concurrent connections
- Real-time message broadcasting
- User join/leave notifications with timestamps
- Thread-based connection handling
- Simple text-based interface

## Target Use Case

Educational/demonstration project for learning TCP socket programming and multi-threaded server architecture. Not intended for production use without security enhancements (no authentication, no encryption, no message persistence).
